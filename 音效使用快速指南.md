## 快速使用

### 1. 在页面中添加点击音效

```typescript
// 1. 导入音效服务
import { soundEffectService } from '../services/SoundEffectService'

// 2. 在onClick中调用
.onClick(() => {
  soundEffectService.playClickSound();
  // 你的业务逻辑
})
```

### 2. 播放背景音乐（仅首页）

```typescript
import { soundEffectService } from '../services/SoundEffectService'
import { common } from '@kit.AbilityKit'

async aboutToAppear() {
  const context = getContext(this) as common.UIAbilityContext;
  await soundEffectService.initialize(context);
  await soundEffectService.playBackgroundMusic('audios/muzhiqin.mp3');
}

async aboutToDisappear() {
  await soundEffectService.stopBackgroundMusic();
}
```

## 常见问题

### Q: 音效不播放？
A: 检查以下几点：
1. 音频文件是否在 `entry/src/main/resources/rawfile/audios/` 目录
2. 是否正确导入 `soundEffectService`
3. 是否在onClick中调用了 `playClickSound()`
4. 检查设备音量是否打开

### Q: 背景音乐不循环？
A: 确保在 `SoundEffectService.ets` 中设置了 `this.bgmPlayer.loop = true`

### Q: 音效和背景音乐冲突？
A: 不会冲突，使用了两个独立的AVPlayer实例

### Q: 如何调整音量？
A: 可以在AVPlayer上设置volume属性：
```typescript
this.bgmPlayer.volume = 0.5; // 0.0 到 1.0
```
