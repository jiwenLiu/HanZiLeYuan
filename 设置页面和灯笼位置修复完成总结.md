### 3. 修复首页灯笼位置 🏮

#### 问题分析
用户反馈"第二个灯笼"位置不对。经过分析：
- 从左到右的顺序：20% → 40% → 60% → 78%
- 第二个灯笼应该是 left: 40% 的那个
- 之前它在 top: 25%（下方），与其他灯笼不在同一行

#### 修复方案
将所有灯笼的top都设置为10%，使它们在同一水平线上。

#### 修复前
```typescript
{ left: '20%', top: '10%', animIndex: 0 },  // 第1个 ✅
{ left: '60%', top: '10%', animIndex: 1 },  // 第3个 ✅
{ left: '40%', top: '25%', animIndex: 2 },  // 第2个 ❌（下方）
{ left: '78%', top: '10%', animIndex: 1 }   // 第4个 ✅
```

#### 修复后
```typescript
{ left: '20%', top: '10%', animIndex: 0 },  // 第1个 ✅
{ left: '40%', top: '10%', animIndex: 2 },  // 第2个 ✅（已对齐）
{ left: '60%', top: '10%', animIndex: 1 },  // 第3个 ✅
{ left: '78%', top: '10%', animIndex: 1 }   // 第4个 ✅
```

#### 视觉效果
```
修复前：
🏮              🏮                    🏮
(20%)          (60%)                (78%)
        🏮
      (40%) ← 下方，不对齐

修复后：
🏮       🏮       🏮           🏮
(20%)   (40%)   (60%)       (78%)
↑       ↑       ↑           ↑
所有灯笼在同一水平线 ✅
```

```typescript

  // 页面内容
}
.width('100%')
.layoutWeight(1)
.indicator(false)        // 隐藏默认指示器
.loop(false)            // 不循环
.duration(300)          // 动画时长
.curve(Curve.EaseInOut) // 缓动曲线
.onChange((index: number) => {
  this.currentPageIndex = index
})
```

```typescript
Row({ space: 8 }) {
```
