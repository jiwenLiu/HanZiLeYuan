### 4. 浮动动画（完全参考启动页）

#### 添加状态变量
```typescript
@State logoTranslateY: number = 0  // 头像浮动动画
private logoTimer: number = -1      // 头像动画定时器
```

#### 动画逻辑
```typescript
// 在 aboutToAppear() 中启动动画
this.logoTimer = setInterval(() => {
  const time = Date.now() / 1000
  this.logoTranslateY = Math.sin(time * 2) * 15
}, 50)
```

#### 应用动画
```typescript
Stack() {
  // ... 头像内容
}
.translate({ y: this.logoTranslateY })
```

#### 清理定时器
```typescript
aboutToDisappear() {
  if (this.logoTimer !== -1) {
    clearInterval(this.logoTimer)
  }
}
```

## 🔧 技术细节

### 渐变色配置
- **起始色**: `#fbbf24` (yellow-400) - 明亮的金黄色
- **结束色**: `#ea580c` (orange-600) - 鲜艳的橙红色
- **渐变角度**: 135° (从左上到右下)

### 阴影效果
- **模糊半径**: 30px
- **颜色**: `rgba(0, 0, 0, 0.3)` (30%透明度黑色)
- **偏移**: X=0, Y=10 (向下偏移)

### 动画参数
- **频率**: 50ms 刷新一次
- **幅度**: ±15px 垂直移动
- **周期**: Math.sin(time * 2) - 约3.14秒一个周期
- **曲线**: 正弦波（平滑上下浮动）
