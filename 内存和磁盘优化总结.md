### 🟡 中期：视频压缩

**工具**: FFmpeg
**目标**: 每个视频压缩到15-20MB

```powershell
# 安装FFmpeg (如果没有)
# 下载: https://ffmpeg.org/download.html

# 批量压缩
$sourceDir = "D:\PC_test\Videos_Full_Backup"
$outputDir = "D:\PC_test\Videos_Compressed"

Get-ChildItem "$sourceDir\*.mp4" | ForEach-Object {
    $output = "$outputDir\$($_.Name)"
    ffmpeg -i $_.FullName `
        -vf scale=1280:720 `
        -c:v libx264 -crf 28 -preset fast `
        -c:a aac -b:a 96k `
        $output
}
```

**参数说明**:
- `scale=1280:720`: 降低分辨率到720p
- `crf 28`: 压缩质量（18-28，数字越大压缩越多）
- `preset fast`: 压缩速度
- `b:a 96k`: 音频码率

**预期效果**:
- 每个视频: 50MB → 15-20MB
- 总大小: 1.4GB → ~500MB
- 画质: 轻微下降，但可接受
- 可恢复29个视频

## 技术优化建议

### 1. LazyForEach（WatchPage）
**当前**: ForEach渲染29个卡片
**优化**: LazyForEach按需渲染

```typescript
// 创建数据源
class VideoDataSource implements IDataSource {
  private videos: VideoInfo[] = []
  
  constructor(videos: VideoInfo[]) {
    this.videos = videos
  }
  
  totalCount(): number {
    return this.videos.length
  }
  
  getData(index: number): VideoInfo {
    return this.videos[index]
  }
  
  registerDataChangeListener(listener: DataChangeListener): void {}
  unregisterDataChangeListener(listener: DataChangeListener): void {}
}

// 使用LazyForEach
LazyForEach(this.videoDataSource, (video: VideoInfo) => {
  this.VideoCard(video)
}, (video: VideoInfo) => video.id)
```

**效果**: 节省约30MB内存
