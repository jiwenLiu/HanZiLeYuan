**关键代码**:
```typescript
export class VideoFileService {
  private externalVideoDir: string = '/storage/emulated/0/hanzileyuan_videos'
  
  getVideoPath(videoFileName: string): string {
    return `${this.externalVideoDir}/${videoFileName}`
  }
  
  checkVideoExists(videoFileName: string): boolean {
    return fs.accessSync(this.getVideoPath(videoFileName))
  }
}
```

  const videoFileName = `第${video.id}课${video.subtitle}.mp4`
  this.videoSrc = this.videoFileService!.getVideoPath(videoFileName)
}

## ⚠️ 注意事项

### 1. 首次部署
- 视频推送需要5-10分钟
- 确保USB连接稳定
- 确保设备有2GB+空闲空间

### 2. 权限管理
- 首次启动必须授予存储权限
- 如果拒绝权限，视频无法播放
- 可在设置中重新授予权限

### 3. 文件路径
- 设备路径: `/storage/emulated/0/hanzileyuan_videos`
- 文件名格式: `第X课《XXX》.mp4`
- 文件名必须完全匹配

### 4. 版本更新
- 应用更新不影响视频文件
- 视频文件可独立更新
- 无需重新推送已有视频
