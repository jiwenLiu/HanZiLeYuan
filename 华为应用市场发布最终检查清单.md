## 📦 构建Release包

### 步骤1：清理项目
```bash
hvigorw clean
```

### 步骤2：构建Release包
```bash
hvigorw assembleApp --mode module -p module=entry@default -p product=default --release
```

### 步骤3：验证HAP包
```bash
# 检查包大小（应该比debug包小20-30%）
ls -lh entry/build/default/outputs/default/entry-default-signed.hap

# 验证签名
hap-sign-tool verify -inFile entry/build/default/outputs/default/entry-default-signed.hap
```

## 📤 提交流程

1. 登录华为开发者联盟：https://developer.huawei.com/
2. 进入"管理中心" > "应用服务" > "AppGallery Connect"
3. 点击"我的应用" > "新建应用"
4. 填写应用信息
5. 上传HAP包
