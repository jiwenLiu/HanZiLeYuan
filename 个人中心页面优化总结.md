### 2. 动画增强 - 第二页

#### ✅ 四个功能卡片添加浮动动画

**动画效果**: 和首页组件一样的平滑浮动

#### 技术实现

##### 添加状态变量
```typescript
@State menuCardOffsets: number[] = [0, 0, 0, 0]  // 四个卡片的浮动偏移
private menuAnimTimer: number = -1  // 动画定时器
```

##### 动画逻辑
```typescript
// 在 aboutToAppear() 中启动
this.menuAnimTimer = setInterval(() => {
  const time = Date.now() / 1000
  for (let i = 0; i < 4; i++) {
    this.menuCardOffsets[i] = Math.sin(time * 1.5 + i * 0.5) * 12
  }
}, 50)
```

##### 应用动画
```typescript
// MenuItem 添加 cardIndex 参数
.translate({ y: this.menuCardOffsets[cardIndex] })
```

##### 清理定时器
```typescript
aboutToDisappear() {
  if (this.menuAnimTimer !== -1) {
    clearInterval(this.menuAnimTimer)
  }
}
```
