## 📋 原型样式分析

### 海底居民页 (sea-creatures.html) 左右按钮

#### HTML/CSS 代码
```html
<!-- 左切换按钮 -->
<button class="nav-button bg-gradient-to-br from-blue-500 to-blue-700 
               w-20 h-20 rounded-full flex items-center justify-center 
               shadow-2xl mr-10">
    <i class="fas fa-chevron-left text-white text-4xl"></i>
</button>

<!-- 右切换按钮 -->
<button class="nav-button bg-gradient-to-br from-blue-500 to-blue-700 
               w-20 h-20 rounded-full flex items-center justify-center 
               shadow-2xl ml-10">
    <i class="fas fa-chevron-right text-white text-4xl"></i>
</button>
```

#### CSS 样式
```css
.nav-button {
    transition: all 0.3s;
}
.nav-button:hover:not(:disabled) {
    transform: scale(1.15);
    box-shadow: 0 12px 30px rgba(59, 130, 246, 0.6);
}
.nav-button:disabled {
    opacity: 0.3;
    cursor: not-allowed;
}
```

#### 样式参数提取
- **尺寸**: w-20 h-20 = 80x80px
- **形状**: rounded-full = 圆形
- **渐变**: from-blue-500 to-blue-700
  - blue-500 = #3b82f6
  - blue-700 = #1d4ed8
- **阴影**: shadow-2xl = 强烈阴影
- **图标**: text-4xl = 48px
- **禁用**: opacity: 0.3

## 🔄 ArkTS 实现

### 左滑动按钮

```typescript
// 左箭头（海底居民样式）
Stack() {
  Column()
    .width(80)              // w-20 = 80px
    .height(80)             // h-20 = 80px
    .borderRadius(40)       // rounded-full
    .linearGradient({
      angle: 135,           // from-blue-500 to-blue-700
      colors: [['#3b82f6', 0], ['#1d4ed8', 1]]
    })
    .shadow({ 
      radius: 30,           // shadow-2xl
      color: 'rgba(59, 130, 246, 0.6)', 
      offsetX: 0,
      offsetY: 12
    })
    .opacity(this.currentPage === 0 ? 0.3 : 1)  // disabled: 0.3

  Text('◀')
    .fontSize(48)           // text-4xl
    .fontColor(Color.White)
}
.onClick(() => {
  this.previousPage()
})
```

### 右滑动按钮

```typescript
// 右箭头（海底居民样式）
Stack() {
  Column()
    .width(80)
    .height(80)
    .borderRadius(40)

      radius: 30,

  Text('▶')
    .fontSize(48)

  this.nextPage()
})
```

## 🔧 技术细节

### 渐变色配置
```typescript
linearGradient({
  angle: 135,  // 从左上到右下
  colors: [
    ['#3b82f6', 0],  // blue-500 起始色
    ['#1d4ed8', 1]   // blue-700 结束色
  ]
})
```

### 阴影配置
```typescript
shadow({ 
  radius: 30,                        // 强烈模糊效果
  color: 'rgba(59, 130, 246, 0.6)',  // 蓝色半透明
  offsetX: 0,                        // 无水平偏移
  offsetY: 12                        // 向下偏移12px
})
```

### 禁用状态
```typescript
// 左按钮：第1页时禁用
.opacity(this.currentPage === 0 ? 0.3 : 1)

// 右按钮：第2页时禁用
.opacity(this.currentPage === 1 ? 0.3 : 1)
```

## 📊 修改前后对比

### 修改前（卡通样式）
```typescript
Column()
  .width(60)              // 较小
  .height(60)
  .borderRadius(30)
  .linearGradient({
    angle: 135,
    colors: [
      ['#3b82f6', 0], 
      ['#2563eb', 0.5],   // 三色渐变
      ['#1d4ed8', 1]
    ]
  })
  .shadow({ 
    radius: 12,           // 较弱阴影
    color: 'rgba(59, 130, 246, 0.5)', 
    offsetX: 0,
    offsetY: 4
  })
  .border({ width: 3, color: 'rgba(255, 255, 255, 0.6)' })  // 有边框
  .opacity(禁用 ? 0.5 : 1)

Text('◀️')                // emoji图标
  .fontSize(28)           // 较小
```

