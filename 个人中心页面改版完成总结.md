#### 页面结构

##### 改版前
- **布局**：竖屏滚动
- **内容**：所有内容在一个页面中上下滚动
- **交互**：上下滑动查看内容

##### 改版后
- **布局**：横屏左右滑动
- **内容**：分为3个独立页面
- **交互**：左右滑动切换页面

## 🔧 技术实现

### 横屏模式
```typescript
async setLandscape() {
  const windowStage = (getContext(this) as common.UIAbilityContext).windowStage
  const mainWindow = await windowStage.getMainWindow()
  await mainWindow.setPreferredOrientation(window.Orientation.LANDSCAPE)
}
```

### Swiper配置
```typescript
Swiper(this.swiperController) {
  this.ProfileInfoPage()      // 第1页
  this.LearningToolsPage()    // 第2页
  this.SettingsPage()         // 第3页
}
.width('100%')
.layoutWeight(1)
.indicator(false)        // 隐藏默认指示器
.loop(false)            // 不循环
.duration(300)          // 动画时长
.curve(Curve.EaseInOut) // 缓动曲线
.onChange((index: number) => {
  this.currentPageIndex = index
})
```

### 自定义指示器
```typescript
Row({ space: 8 }) {
  ForEach([0, 1, 2], (index: number) => {
    Column()
      .width(this.currentPageIndex === index ? 24 : 8)
      .height(8)
      .borderRadius(4)
      .backgroundColor(
        this.currentPageIndex === index 
          ? Color.White 
          : 'rgba(255, 255, 255, 0.3)'
      )
      .animation({
        duration: 300,
        curve: Curve.EaseInOut
      })
  })
}
```

### 页面生命周期
```typescript
async aboutToAppear() {
  await this.setFullScreen()   // 全屏模式
  await this.setLandscape()    // 横屏模式
}
```
