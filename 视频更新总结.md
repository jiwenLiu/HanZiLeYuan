## ID映射关系

所有页面的视频 ID 都使用课程编号（'1', '2', '3'...），与文件名中的课程编号一致。

播放时通过以下方式生成文件路径：
```typescript
this.videoSrc = `videos/第${video.id}课${video.subtitle}.mp4`
```

例如：
- ID='1' → `videos/第1课《马牛羊鸡犬豕》.mp4`
- ID='27' → `videos/第27课《父男兄弟叔舅》.mp4`
