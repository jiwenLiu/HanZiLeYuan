#### 创建防沉迷服务
**文件**: `entry/src/main/ets/services/AntiAddictionService.ets`

**核心功能**:
- ✅ 监控应用使用时长
- ✅ 提前5分钟提醒
- ✅ 达到时长限制时弹出对话框
- ✅ 自动返回首页
- ✅ 持久化存储设置

**技术实现**:
```typescript
class AntiAddictionService {
  - 每分钟检查一次使用时长
  - 提前5分钟显示警告提示
  - 达到限制时显示对话框并返回首页
  - 支持自定义时长设置
  - 数据持久化存储
}
```

#### 集成到应用
- **HomePage**: 启动防沉迷监控
- **ProfilePage**: 设置防沉迷时长

**使用流程**:
1. 用户在个人中心设置防沉迷时长（默认30分钟）
2. 进入首页时自动启动监控
3. 提前5分钟显示提醒Toast
4. 达到时长后显示对话框
5. 用户点击"知道了"后返回首页

## 📊 技术细节

### 防沉迷服务API

#### 初始化
```typescript
await antiAddictionService.initialize(context)
```

#### 启动监控
```typescript
antiAddictionService.startMonitoring()
```

#### 停止监控
```typescript
antiAddictionService.stopMonitoring()
```

#### 设置时长
```typescript
await antiAddictionService.saveSettings(minutes)
```

#### 获取当前设置
```typescript
const minutes = antiAddictionService.getLimitMinutes()
```

#### 获取已使用时长
```typescript
const used = antiAddictionService.getUsedMinutes()
```

#### 重置计时器
```typescript
antiAddictionService.reset()
```

### 数据持久化

**存储位置**: `preferences`  
**存储键**: `limitMinutes`  
**默认值**: 30分钟

**存储逻辑**:
```typescript
// 保存
await dataPreferences.put('limitMinutes', minutes)
await dataPreferences.flush()

// 读取
const minutes = await dataPreferences.get('limitMinutes', 30)
```
