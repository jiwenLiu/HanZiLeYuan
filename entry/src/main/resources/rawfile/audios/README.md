# 音频资源文件说明

## 目录结构
```
entry/src/main/resources/rawfile/audios/
├── ma.mp3      # 马
├── ji.mp3      # 鸡
├── niao.mp3    # 鸟
├── mu.mp3      # 木
├── shu.mp3     # 树
├── mi.mp3      # 米
├── shui.mp3    # 水
├── he.mp3      # 河
├── shi.mp3     # 石
├── ren.mp3     # 人
├── da.mp3      # 大
├── zi.mp3      # 子
├── yi.mp3      # 衣
├── jin.mp3     # 巾
└── bei.mp3     # 贝
```

## 音频要求
- 格式：MP3
- 编码：AAC
- 采样率：44100Hz
- 比特率：128kbps
- 时长：2-3秒
- 内容：清晰的汉字发音

## 使用说明
1. 将对应的汉字发音音频文件放入此目录
2. 确保文件名与上述列表完全一致
3. 音频应包含标准普通话发音
4. 建议使用专业的TTS工具或录音生成

## 临时解决方案
如果暂时没有音频文件，可以：
1. 使用在线TTS服务生成音频
2. 使用录音软件录制发音
3. 从其他项目复制类似的音频文件

## 注意事项
- 音频文件不应过大（建议每个文件小于100KB）
- 确保音频清晰无杂音
- 音频开头和结尾不应有过长的静音
