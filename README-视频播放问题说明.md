# 视频播放问题说明

## 当前问题

应用报错：`错误码103602 - Not a valid source`

## 根本原因

经过多次测试，确认问题原因：

1. **HarmonyOS Video组件的限制**
   - rawfile方式：仅支持小文件（推荐<30MB）
   - 本地文件路径：Video组件不支持沙箱目录的绝对路径
   - 当前视频文件：40-70MB，超过rawfile限制

2. **已尝试的方案**
   - ✅ rawfile + $rawfile() - 文件太大，无法播放
   - ✅ 沙箱目录 + 绝对路径 - Video组件不支持
   - ✅ 沙箱目录 + file://协议 - Video组件不支持
   - ✅ fd://协议 - Video组件不支持

## 最终解决方案

**压缩视频文件到20-30MB以内**

详细步骤请参考：`视频压缩指南.md`
