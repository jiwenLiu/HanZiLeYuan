2025年12月2日

## 问题1：应用图标修改后没有效果

### 诊断结果
✅ **图标文件正常**

- AppScope图标：存在，大小 49,082 字节
- Entry图标：存在，大小 260,622 字节

### 问题原因
图标文件本身没有问题，但是修改后没有效果的原因是：

**系统缓存了旧图标**

HarmonyOS系统会缓存应用图标，即使重新安装，如果包名相同，系统可能仍然显示旧图标。

### 解决方案

#### 方法1：完全卸载后重新安装（推荐）

```bash
# 步骤1：卸载应用
hdc uninstall com.chinasoft.tools.hanzileyuan

# 步骤2：清理构建缓存
hvigorw clean

# 步骤3：重新编译
hvigorw assembleHap

# 步骤4：安装应用
hdc install entry\build\default\outputs\default\entry-default-signed.hap
```

#### 方法2：修改版本号强制更新

修改 `AppScope/app.json5`：

```json
{
  "app": {
    "versionCode": 1000001,  // 从1000000改为1000001
    "versionName": "1.0.1",  // 从1.0.0改为1.0.1
    ...
  }
}
```

然后重新编译安装。

#### 方法3：重启设备

有时候重启设备可以清除图标缓存。

- ✗ yi.mp3 - 文件为空！

- ✗ bei.mp3 - 文件为空！

### 问题原因
当前的音频文件是在创建项目时生成的占位文件（0字节），没有实际的音频内容。

`AudioPlayService` 尝试播放这些空文件时会失败，所以没有声音。

### 解决方案

#### 推荐方案：使用在线TTS生成音频

##### 1. 百度语音合成（推荐）
- 网址：https://ai.baidu.com/tech/speech/tts
- 优点：免费，音质好，支持多种发音人
- 步骤：
  1. 注册百度AI账号
  2. 进入语音合成页面
  3. 输入汉字（如：马）
  4. 选择发音人（建议：标准女声或童声）
  5. 下载MP3文件
  6. 重命名为 `ma.mp3`

##### 2. 讯飞语音合成
- 网址：https://www.xfyun.cn/services/online_tts
- 优点：音质清晰，支持多种方言
- 步骤类似百度

##### 3. 阿里云语音合成
- 网址：https://ai.aliyun.com/nls/tts
- 优点：专业级音质
- 步骤类似百度

#### 音频文件要求

- **格式**：MP3
- **编码**：AAC
- **采样率**：44100Hz
- **比特率**：128kbps
- **时长**：2-3秒
- **内容**：只需要汉字的发音，不需要拼音和释义
- **文件大小**：建议每个文件小于100KB

#### 替换步骤

1. **生成所有音频文件**
   - 使用在线TTS逐个生成15个汉字的发音
   - 下载为MP3格式

2. **重命名文件**
   - 确保文件名与上表完全一致
   - 例如："马"的发音文件必须命名为 `ma.mp3`

3. **复制到项目**
   ```
   将15个音频文件复制到：
   entry/src/main/resources/rawfile/audios/
   
   覆盖现有的空文件
   ```

4. **验证文件**
   - 运行 `check_and_fix.bat` 再次检查
   - 确保所有文件大小都不为0

5. **重新编译安装**
   ```bash
   hvigorw assembleHap
   hdc install entry\build\default\outputs\default\entry-default-signed.hap
   ```

6. **测试功能**
   - 打开应用
   - 进入"听一听"页面
   - 点击播放按钮
   - 应该能听到汉字发音

## 总结

- **状态**：图标文件正常

- **解决**：完全卸载后重新安装
