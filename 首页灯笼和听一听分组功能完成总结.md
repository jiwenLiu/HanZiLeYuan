### 2. 听一听页面分组功能 🎧

#### 功能概述
- 将48个汉字MP3分成4组，每组15个
- 每次进入页面自动显示一组未播放过的汉字
- 播放完当前组的最后一个汉字后，自动标记该组为已播放
- 所有组播放完后，自动重置为第一组

#### 技术实现

##### 2.1 数据结构
```typescript
@State currentGroupIndex: number = 0  // 当前显示的组索引（0-3）
@State displayAudioList: AudioInfo[] = []  // 当前显示的15个汉字

private dataPreferences: preferences.Preferences | null = null
private readonly GROUP_SIZE: number = 15  // 每组15个
private readonly TOTAL_GROUPS: number = 4  // 总共4组
private allAudioList: AudioInfo[] = [...]  // 完整的48个汉字
```

##### 2.2 分组逻辑
- **第1组**：汉字1-15（动物篇1-4 + 植物篇5-7 + 自然篇8-15）
- **第2组**：汉字16-30（自然篇16-22 + 身体篇23-30）
- **第3组**：汉字31-45（身体篇31-42 + 日常篇43-45）
- **第4组**：汉字46-48（日常篇46-48，共3个）

##### 2.3 核心功能

**初始化流程**
```typescript
async aboutToAppear() {
  await this.setFullScreen()
  await audioPlayService.initialize()
  await this.initDataPreferences()  // 初始化数据存储
  await this.loadCurrentGroup()     // 加载当前组
}
```

**加载分组**
```typescript
async loadCurrentGroup() {
  // 1. 读取已播放的组
  const playedGroupsStr = await this.dataPreferences.get('played_groups', '[]')
  const playedGroups: number[] = JSON.parse(playedGroupsStr)
  
  // 2. 找到第一个未播放的组
  let targetGroupIndex = 0
  for (let i = 0; i < this.TOTAL_GROUPS; i++) {
    if (!playedGroups.includes(i)) {
      targetGroupIndex = i
      break
    }
  }
  
  // 3. 如果所有组都播放过了，重置为第一组
  if (playedGroups.length >= this.TOTAL_GROUPS) {
    targetGroupIndex = 0
    await this.dataPreferences.put('played_groups', '[]')
    await this.dataPreferences.flush()
  }
  
  // 4. 加载对应组的汉字
  this.currentGroupIndex = targetGroupIndex
  this.loadGroupAudios()
}
```

**标记已播放**
```typescript
async playCurrentAudio() {
  const currentAudio = this.displayAudioList[this.currentIndex]
  if (currentAudio && currentAudio.audioPath) {
    this.isPlaying = true
    await audioPlayService.playAudio(currentAudio.audioPath)
    
    // 如果是当前组的最后一个汉字，标记该组为已播放
    if (this.currentIndex === this.displayAudioList.length - 1) {
      await this.markGroupAsPlayed()
    }
    
    setTimeout(() => {
      this.isPlaying = false
    }, 2000)
  }
}
```

## 📱 使用说明

### 用户使用流程
1. **首次进入**：自动显示第1组（15个汉字）
2. **学习完成**：播放到第15个汉字时，自动标记第1组为已播放
3. **再次进入**：自动显示第2组（15个汉字）
4. **循环学习**：所有4组播放完后，自动重置为第1组

### 进度查看
- 页面右侧显示：`第X组 Y/Z`
  - X：当前组号（1-4）
  - Y：当前汉字序号（1-15）
  - Z：当前组总汉字数（15或3）

### 控制按钮
- **⬅️ 上一个**：播放上一个汉字（到达第一个时禁用）
- **▶️ 播放/暂停**：播放或暂停当前汉字音频
- **➡️ 下一个**：播放下一个汉字（到达最后一个时禁用）
