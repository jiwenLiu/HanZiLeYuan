## 🎨 首页动画效果

### 核心动画代码

```css
/* 卡片漂浮动画 */
@keyframes card-float {
    0%, 100% { transform: translateY(0); }
    50% { transform: translateY(-15px); }
}

/* 卡片基础样式 */
.card {
    transition: all 0.3s;
    animation: card-float ease-in-out infinite;
    opacity: 0.85;
    backdrop-filter: blur(10px);
}

/* 悬停效果 */
.card:hover {
    animation-play-state: paused;
    transform: translateY(-8px) scale(1.02);
    opacity: 1;
    box-shadow: 0 20px 50px rgba(0,0,0,0.3);
}

/* 不同卡片的动画延迟 */
.card:nth-child(1) {
    animation-duration: 3s;
    animation-delay: 0s;
}

.card:nth-child(2) {
    animation-duration: 3.5s;
    animation-delay: 0.2s;
}

.card:nth-child(3) {
    animation-duration: 3.2s;
    animation-delay: 0.4s;
}

.card:nth-child(4) {
    animation-duration: 3.8s;
    animation-delay: 0.6s;
}
```

## 📐 动画效果详解

### 1. 漂浮动画 ✅

**动画名称**：`card-float`

**效果描述**：
- 卡片在垂直方向上下漂浮
- 幅度：15px
- 循环播放，无限重复

**动画参数**：
```css
animation: card-float ease-in-out infinite;
```

**关键帧**：
```
0%   → Y轴位置: 0px    (起始位置)
50%  → Y轴位置: -15px  (向上移动)
100% → Y轴位置: 0px    (回到起始)
```

**视觉效果**：
```
    ↑ -15px
    │
────┼──── 0px (起始)
    │
    ↓
```

---

### 2. 透明度效果 ✅

**默认透明度**：`opacity: 0.85`
- 轻微半透明
- 营造层次感
- 磨砂玻璃效果

**悬停透明度**：`opacity: 1`
- 完全不透明
- 突出选中状态
- 清晰显示内容

---

### 3. 模糊背景 ✅

**背景模糊**：`backdrop-filter: blur(10px)`
- 磨砂玻璃效果
- 背景虚化
- 现代化设计
- 增强层次感

---

### 4. 悬停动画 ✅

**鼠标悬停时**：
```css
.card:hover {
    animation-play-state: paused;  /* 暂停漂浮 */
    transform: translateY(-8px) scale(1.02);  /* 上移+放大 */
    opacity: 1;  /* 完全不透明 */
    box-shadow: 0 20px 50px rgba(0,0,0,0.3);  /* 增强阴影 */
}
```

**效果分解**：
1. **暂停漂浮**：停止上下漂浮
2. **向上移动**：上移8px
3. **轻微放大**：放大到1.02倍
4. **完全不透明**：opacity从0.85变为1
5. **增强阴影**：阴影扩大，增强立体感

---

### 5. 动画延迟 ✅

**交错延迟效果**：

| 卡片序号 | 动画时长 | 延迟时间 | 效果 |
|----------|----------|----------|------|
| 第1个 | 3.0s | 0.0s | 最先开始 |
| 第2个 | 3.5s | 0.2s | 稍后开始 |
| 第3个 | 3.2s | 0.4s | 再稍后 |
| 第4个 | 3.8s | 0.6s | 最后开始 |

**视觉效果**：
```
卡片1: ↑↓  (3.0s)
卡片2:  ↑↓ (3.5s, 延迟0.2s)
卡片3:   ↑↓ (3.2s, 延迟0.4s)
卡片4:    ↑↓ (3.8s, 延迟0.6s)
```

**优点**：
- ✅ 波浪式漂浮

## 💡 技术细节

### CSS关键技术

#### 1. CSS动画（@keyframes）

```css
@keyframes card-float {
    0%, 100% { transform: translateY(0); }
    50% { transform: translateY(-15px); }
}
```

**作用**：定义漂浮动画轨迹

---

#### 2. 动画属性

```css
animation: card-float ease-in-out infinite;
```

**参数解析**：
- `card-float`：动画名称
- `ease-in-out`：缓动函数（先慢后快再慢）
- `infinite`：无限循环

---

#### 3. 动画控制

```css
animation-play-state: paused;
```

**作用**：鼠标悬停时暂停动画

---

#### 4. nth-child选择器

```css
.card:nth-child(1) {
    animation-duration: 3s;
    animation-delay: 0s;
}
```

**作用**：为不同位置的卡片设置不同的动画参数

---

#### 5. backdrop-filter

```css
backdrop-filter: blur(10px);
```

**作用**：背景模糊效果（磨砂玻璃）

---

#### 6. transform组合

```css
transform: translateY(-8px) scale(1.02);
```

**作用**：同时应用位移和缩放
