# ✅ 成就中心筛选功能更新说明

## 🔄 功能变更

### 之前的设计
- 点击"学习类"、"练习类"、"挑战类"、"特殊类" → 跳转到单独页面
- 每个分类有独立的HTML页面
- 需要返回按钮才能回到主页

### 现在的设计 ✅
- 点击分类按钮 → **在当前页面内筛选显示**
- 无需跳转，即时响应
- 统计数据实时更新
- 更流畅的用户体验

---

## 🎯 筛选功能

### 分类按钮
- **全部**：显示所有成就（默认）
- **学习类**：显示学习相关成就
- **练习类**：显示练习相关成就
- **挑战类**：显示挑战相关成就
- **特殊类**：显示特殊成就

### 按钮状态
- **选中状态**：`bg-white/30`（较亮背景）
- **未选中状态**：`bg-white/10`（较暗背景）
- **悬停效果**：`hover:bg-white/20`

---

## 📊 统计数据更新

点击不同分类时，顶部统计卡片会实时更新：
- **已获得**：当前分类下已解锁的成就数
- **进行中**：当前分类下正在进行的成就数
- **总数量**：当前分类下的成就总数

---

## 🏷️ 成就分类标记

每个成就卡片都添加了 `data-category` 属性：

## 💻 技术实现

### 1. 筛选按钮HTML
```html
<div class="filter-btn bg-white/30 text-white px-4 py-2 rounded-full" 
     data-category="all" 
     onclick="filterAchievements('all')">
    全部
</div>
```

### 2. 筛选函数
```javascript
function filterAchievements(category) {
    // 1. 更新按钮选中状态
    const filterBtns = document.querySelectorAll('.filter-btn');
    filterBtns.forEach(btn => {
        if (btn.getAttribute('data-category') === category) {
            btn.classList.add('bg-white/30');  // 选中
        } else {
            btn.classList.remove('bg-white/30');
            btn.classList.add('bg-white/10');  // 未选中
        }
    });
    
    // 2. 显示/隐藏成就卡片
    const achievements = document.querySelectorAll('.achievement-card');
    achievements.forEach(card => {
        const cardCategory = card.getAttribute('data-category');
        if (category === 'all' || cardCategory === category) {
            card.style.display = 'block';  // 显示
        } else {
            card.style.display = 'none';   // 隐藏
        }
    });
    
    // 3. 更新统计数据
    updateStats(category);
}
```

### 3. 统计更新函数
```javascript
function updateStats(category) {
    let unlocked = 0;    // 已获得
    let inProgress = 0;  // 进行中
    let total = 0;       // 总数
    
    // 遍历可见成就
    achievements.forEach(card => {
        if (category === 'all' || card.getAttribute('data-category') === category) {
            total++;
            if (card.classList.contains('unlocked')) {
                unlocked++;
            } else if (!card.classList.contains('locked')) {
                inProgress++;
            }
        }
    });
    
    // 更新显示
    // ...
}
```
