## ✨ 交互效果

### 1. 悬停效果
```css
transform: translateY(-10px) scale(1.05);  /* 上移+放大 */
opacity: 1;                                 /* 不透明度100% */
box-shadow: 0 20px 40px rgba(0,0,0,0.4);  /* 阴影增强 */
```

### 2. 浮动动画
- 每个图标都有浮动效果
- 上下移动15px
- 3秒一个周期
- 每个卡片延迟不同（0s, 0.2s, 0.4s, 0.6s, 0.8s, 1s）

```css
@keyframes float {
    0%, 100% { transform: translateY(0px); }
    50% { transform: translateY(-15px); }
}
```

### 3. 返回按钮动画
- 淡入缩放动画
- 左右弹跳效果
- 悬停时金色阴影

## 📱 响应式设计

### 滚动支持
- `overflow-auto`：内容超出时自动滚动
- `min-h-screen`：最小高度为视口高度
- `pb-8`：底部留白32px

### 布局适配
- **桌面端**：3列网格，每列等宽
- **平板端**：3列网格（保持）
- **手机端**：建议改为2列或1列（未来优化）

## 🚀 使用方法

```
index.html → 点击"课程探索"快捷按钮 → explore.html
```

### 2. 选择分类
在explore.html页面中：
- 浏览6个分类卡片
- 悬停查看交互效果
- 点击进入对应分类

- 点击左上角返回按钮
- 或通过浏览器后退
