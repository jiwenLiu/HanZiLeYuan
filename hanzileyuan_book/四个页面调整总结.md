# 四个页面调整总结

## 🎯 调整目标

将植物篇、自然篇、人体篇、日常篇四个页面按照动物篇的样式和布局进行统一调整。

---

## 📊 调整内容

### 统一的设计特点 ✅

#### **布局方式**
- **居中布局**: 使用 `flex items-center justify-center min-h-screen` 将内容垂直居中
- **上移偏移**: `margin-top: -40px` 使内容略微向上，与首页对齐
- **3列网格**: `grid grid-cols-3 gap-6` 每行显示3个课程

#### **翻页功能**
- **左右箭头**: 56×56px 圆形按钮，渐变背景 (`bg-gradient-to-br from-yellow-400 to-orange-500`)
- **箭头间距**: 与卡片区域间距 32px (`gap-8`)
- **禁用状态**: 首页左箭头、末页右箭头 50%透明度 + 禁用鼠标

#### **页码指示器**
- **圆点样式**: 12×12px 圆点 (`w-3 h-3 rounded-full`)
- **颜色**: 当前页白色 (`bg-white`)，其他页 50%透明度 (`bg-white/50`)
- **位置**: 卡片下方，居中对齐

#### **卡片样式**
| 元素 | 样式 | 说明 |
|------|------|------|
| **容器** | `rounded-3xl p-6` | 24px圆角,24px内边距 |
| **图标** | `text-6xl text-center mb-3` | 60px，居中 |
| **标题** | `text-xl font-bold text-center mb-1.5` | 20px，居中 |
| **描述** | `text-xs text-center mb-2` | 12px，居中 |
| **时长** | `text-xs text-center mb-2` | ⏱️ + 时间 |
| **状态** | `text-xs px-2 py-1 rounded` | 已完成/学习中/未开始 |
| **星级** | `<i class="fas fa-star"></i>` | ⭐⭐⭐ |

## 📐 技术实现

### HTML结构

```html
<div class="flex items-center justify-center min-h-screen px-6" style="margin-top: -40px;">
    <div class="w-full max-w-5xl">
        <!-- 标题 -->
        <div class="text-center mb-6">
            <h1>...</h1>
            <p>...</p>
        </div>
        
        <!-- 卡片容器 + 箭头 -->
        <div class="flex items-center gap-8">
            <!-- 左箭头 -->
            <button id="prevBtn" onclick="previousPage()">...</button>
            
            <!-- 卡片区域 -->
            <div class="flex-1 relative overflow-hidden">
                <div id="page1" class="grid grid-cols-3 gap-6">
                    <!-- 3个卡片 -->
                </div>
                <div id="page2" class="grid grid-cols-3 gap-6 hidden">
                    <!-- 3个卡片 -->
                </div>
            </div>
            
            <!-- 右箭头 -->
            <button id="nextBtn" onclick="nextPage()">...</button>
        </div>
        
        <!-- 页码指示器 -->
        <div class="flex justify-center gap-2 mt-6">
            <div id="dot1" class="w-3 h-3 rounded-full bg-white"></div>
            <div id="dot2" class="w-3 h-3 rounded-full bg-white/50"></div>
        </div>
    </div>
</div>
```

### JavaScript翻页逻辑

```javascript
let currentPage = 1;
const totalPages = 5; // 根据实际页数调整

function updatePageDisplay() {
    // 1. 隐藏所有页面
    for (let i = 1; i <= totalPages; i++) {
        document.getElementById('page' + i).classList.add('hidden');
        document.getElementById('dot' + i).classList.remove('bg-white');
        document.getElementById('dot' + i).classList.add('bg-white/50');
    }
    
    // 2. 显示当前页面
    document.getElementById('page' + currentPage).classList.remove('hidden');
    document.getElementById('dot' + currentPage).classList.add('bg-white');
    
    // 3. 更新箭头状态
    if (currentPage === 1) {
        prevBtn.classList.add('opacity-50', 'cursor-not-allowed');
    } else {
        prevBtn.classList.remove('opacity-50', 'cursor-not-allowed');
    }
    
    if (currentPage === totalPages) {
        nextBtn.classList.add('opacity-50', 'cursor-not-allowed');
    } else {
        nextBtn.classList.remove('opacity-50', 'cursor-not-allowed');
    }
}

function previousPage() {
    if (currentPage > 1) {
        currentPage--;
        updatePageDisplay();
    }
}

function nextPage() {
    if (currentPage < totalPages) {
        currentPage++;
        updatePageDisplay();
    }
}

// 初始化
updatePageDisplay();
```
