# 首页四个组件单行布局调整说明

## 🎯 调整目标

将首页的四个漂浮组件从 **2x2 网格布局** 调整为 **1x4 单行布局**，宽度缩短一半。

## 🎯 设计原理

### 1. 宽度缩短策略

**目标**: 宽度缩短一半

**实现方式**:
- ✅ 列数翻倍: 2列 → 4列
- ✅ 容器扩宽: 768px → 1024px
- ✅ 实际效果: 单个卡片宽度从 ~350px 缩减到 ~240px

**计算**:
```
之前: (768px - 16px×1) / 2 = 376px
现在: (1024px - 12px×3) / 4 = 247px
缩减比例: (376 - 247) / 376 = 34.3%
```

### 2. 比例协调

所有内部元素按比例缩小，保持视觉平衡：

| 元素 | 缩减幅度 | 原因 |
|------|---------|------|
| 内边距 | 40% | 最大缩减，节省空间 |
| 标题 | 20% | 保持可读性 |
| 图标 | 17% | 保持识别度 |
| 描述 | 14% | 适度缩小 |

### 3. 响应式考虑

```css
/* 自动适应屏幕宽度 */
.grid-cols-4 {
    grid-template-columns: repeat(4, minmax(0, 1fr));
}

/* 最大宽度限制 */
.max-w-5xl {
    max-width: 1024px; /* 64rem */
}
```

## 🔧 技术实现细节

### Flexbox + Grid 组合

```html
<!-- 外层 Flexbox: 垂直居中 -->
<div class="flex items-center justify-center min-h-screen">
    
    <!-- 内层 Grid: 4列布局 -->
    <div class="grid grid-cols-4 gap-3 max-w-5xl">
        <!-- 4个卡片 -->
    </div>
    
</div>
```

### 响应式网格

```css
/* 自动等宽4列 */
.grid-cols-4 {
    grid-template-columns: repeat(4, minmax(0, 1fr));
}

/* 每列最小0，最大1fr (弹性单位) */
/* 自动填充可用空间 */
```

### 居中偏移

```css
margin-top: -40px;  /* 向上偏移，避免过于正中 */
```
