# 课程探索页翻页功能说明

## 🎯 功能目标

在课程探索页实现左右翻页功能：
1. ✅ 第一页显示前三个组件（动物篇、植物篇、自然篇）
2. ✅ 第二页显示后三个组件（人体篇、日常篇、全部课程）
3. ✅ 左边显示 < 箭头，右边显示 > 箭头
4. ✅ 点击 > 切换到第二页，点击 < 返回第一页
5. ✅ 箭头样式与布局协调适配

---

## 📊 功能实现

### 2. 页面分组

#### **第一页（默认显示）**
- 🐴 动物篇 (4节课)
- 🌱 植物篇 (3节课)
- ☀️ 自然篇 (15节课)

#### **第二页**
- 👨 人体篇 (20节课)
- 👕 日常篇 (6节课)
- 📚 全部课程 (48节课)

---

### 3. 交互逻辑

#### **左箭头 (<)**
- **功能**: 返回第一页
- **状态**: 
  - 在第一页时：灰色半透明 (opacity-50)，不可点击
  - 在第二页时：正常显示，可点击

#### **右箭头 (>)**
- **功能**: 进入第二页
- **状态**:
  - 在第一页时：正常显示，可点击
  - 在第二页时：灰色半透明 (opacity-50)，不可点击

#### **页码指示器**
- 第一页：● ○ (第一个点高亮)
- 第二页：○ ● (第二个点高亮)

## 💡 JavaScript 实现

### 1. 核心变量

```javascript
let currentPage = 1;  // 当前页码，1 或 2
```

---

### 2. 更新显示函数

```javascript
function updatePageDisplay() {
    const page1 = document.getElementById('page1');
    const page2 = document.getElementById('page2');
    const prevBtn = document.getElementById('prevBtn');
    const nextBtn = document.getElementById('nextBtn');
    const dot1 = document.getElementById('dot1');
    const dot2 = document.getElementById('dot2');

    if (currentPage === 1) {
        // 显示第一页
        page1.classList.remove('hidden');
        page2.classList.add('hidden');
        
        // 左箭头禁用，右箭头启用
        prevBtn.classList.add('opacity-50', 'cursor-not-allowed');
        nextBtn.classList.remove('opacity-50', 'cursor-not-allowed');
        
        // 第一个点高亮
        dot1.classList.add('bg-white');
        dot1.classList.remove('bg-white/40');
        dot2.classList.add('bg-white/40');
        dot2.classList.remove('bg-white');
    } else {
        // 显示第二页
        page1.classList.add('hidden');
        page2.classList.remove('hidden');
        
        // 左箭头启用，右箭头禁用
        prevBtn.classList.remove('opacity-50', 'cursor-not-allowed');
        nextBtn.classList.add('opacity-50', 'cursor-not-allowed');
        
        // 第二个点高亮
        dot1.classList.add('bg-white/40');
        dot1.classList.remove('bg-white');
        dot2.classList.add('bg-white');
        dot2.classList.remove('bg-white/40');
    }
}
```

**功能**：
- 切换页面显示/隐藏
- 更新箭头启用/禁用状态
- 更新页码指示器高亮

---

### 3. 翻页函数

```javascript
// 上一页
function previousPage() {
    if (currentPage > 1) {
        currentPage = 1;
        updatePageDisplay();
    }
}

// 下一页
function nextPage() {
    if (currentPage < 2) {
        currentPage = 2;
        updatePageDisplay();
    }
}
```

**逻辑**：
- 检查是否可以翻页
- 更新页码
- 调用更新显示函数

---

### 4. 初始化

```javascript
// 页面加载时初始化显示
updatePageDisplay();
```
