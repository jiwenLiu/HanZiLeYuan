# 🏮 汉字乐园 - HarmonyOS儿童教育App原型

## 📖 项目简介

**汉字乐园**是一款专为3-8岁儿童设计的汉字学习应用，采用HarmonyOS设计规范，参照深海探险家原型风格打造，提供沉浸式的汉字学习体验。

### 🎯 核心特色

- ✅ **完整符合HarmonyOS设计规范** - 按照华为设计语言打造
- ✅ **底部TabBar导航** - 4个核心Tab，流畅切换
- ✅ **48个视频课程** - 动物篇、植物篇、自然篇各16个
- ✅ **互动练习功能** - 听、写、答全方位练习
- ✅ **精美动画效果** - 20+种背景动画元素
- ✅ **手写识别练习** - Canvas实现真实书写体验
- ✅ **可直接用于开发** - 代码结构清晰，模块化设计

## 💡 技术实现

- **Canvas API** - 手写功能实现

### iframe通信机制

```javascript
// 子页面向父页面发送消息
window.parent.postMessage({
    action: 'navigate',
    page: 'learning.html'
}, '*');

// 父页面监听消息
window.addEventListener('message', function(e) {
    if (e.data.action === 'navigate') {
        document.getElementById('mainFrame').src = e.data.page;
    }
});
```

### TabBar切换逻辑

```javascript
function switchTab(page) {
    // 发送切换消息
    window.parent.postMessage({
        action: 'navigate',
        page: page
    }, '*');
    
    // 更新Tab状态
    document.querySelectorAll('.tab-item').forEach(item => {
        item.classList.remove('active');
    });
    event.currentTarget.classList.add('active');
}
```
