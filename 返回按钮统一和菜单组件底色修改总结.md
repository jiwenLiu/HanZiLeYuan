## 🎨 任务1: 返回按钮样式统一

### 个人中心返回按钮样式（标准）
```typescript
Stack() {
  Column()
    .width(56)
    .height(56)
    .borderRadius(28)
    .linearGradient({
      angle: 135,
      colors: [['#fbbf24', 0], ['#f97316', 1]]  // 黄色 → 橙色
    })
    .shadow({
      radius: 20,
      color: 'rgba(0, 0, 0, 0.3)',
      offsetX: 0,
      offsetY: 4
    })
  
  Text('←')
    .fontSize(28)
    .fontColor(Color.White)
}
```

### 关键属性对比

| 属性 | 原始样式 | 个人中心样式 | 变化 |
|------|---------|-------------|------|
| 宽度 | 52px | **56px** | +4px |
| 高度 | 52px | **56px** | +4px |
| 圆角 | 26px | **28px** | +2px |
| 渐变起始色 | #fbbf24 | **#fbbf24** | 相同 |
| 渐变结束色 | #f59e0b | **#f97316** | 更橙 |
| 阴影半径 | 25px | **20px** | -5px |
| 阴影颜色 | rgba(0,0,0,0.4) | **rgba(0,0,0,0.3)** | 更淡 |
| 阴影偏移Y | 2px | **4px** | +2px |
| 箭头字号 | 32px | **28px** | -4px |
| 箭头粗细 | Bold | **Normal** | 更细 |

## 🎨 设计理念

### 统一性
- **原则**: 所有返回按钮使用相同样式

- **优势**: 降低学习成本，提升辨识度

- **原则**: 菜单组件使用主题黄色
- **效果**: 与应用整体色调协调
- **优势**: 视觉更温暖，不刺眼

- **原则**: 细节优化，追求精致
- **效果**: 按钮更大、箭头更细、阴影更柔和
- **优势**: 整体质感提升
