## 🔄 左箭头修改详情

### 修改前（文本样式）
```typescript
Stack() {
  Column()
    .width(48)
    .height(48)
    .borderRadius(24)
    .linearGradient({
      angle: 135,
      colors: [['#f97316', 0], ['#ea580c', 1]]
    })
    .shadow({ radius: 30, color: 'rgba(0,0,0,0.4)', offsetY: 2 })

  Text('‹')
    .fontSize(48)
    .fontColor(Color.White)
    .fontWeight(FontWeight.Bold)
}
.opacity(this.currentPage > 0 ? 1 : 0.3)
```

### 修改后（图标样式）
```typescript
Image($r('app.media.ic_arrow_left'))
  .width(80)
  .height(80)
  .opacity(this.currentPage > 0 ? 1 : 0.3)
  .shadow({ 
    radius: 30, 
    color: 'rgba(249, 115, 22, 0.6)',
    offsetX: 0,
    offsetY: 12
  })
  .onClick(() => {
    this.previousPage()
  })
```
